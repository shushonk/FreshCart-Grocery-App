# FreshCart Grocery App 🛒

FreshCart is a modern, premium grocery shopping application built with Node.js and Express. It features a sleek user interface, dynamic product listings, and a seamless cart management system.

## ✨ Features
- **Premium Design**: Vibrant and responsive UI for a superior shopping experience.
- **Dynamic Product Catalog**: Browse through a wide variety of fresh groceries.
- **Interactive Shopping Cart**: Add, remove, and manage items in your cart in real-time.
- **Order Management**: Robust backend to handle customer orders and data persistence.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shushonk/FreshCart-Grocery-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd FreshCart-Grocery-App
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server:
```bash
npm start
```
The app will be available at `http://localhost:3000`.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Backend**: Node.js, Express.js
- **Data Persistence**: JSON-based storage (extendable to MongoDB)

## 📁 Project Structure
- `server.js`: Main entry point for the Express server.
- `routes/`: API routes for products and cart operations.
- `models/`: Data models and logic.
- `public/`: Static assets (HTML, CSS, JS, Images).
- `data/`: JSON data files for products and persistence.
